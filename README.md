# springboot_backend---POS-System
Spring Boot Backend Project
This is a simple backend project built using Spring Boot. It provides APIs for managing Customers, Items, and Orders.
I am a student and currently learning Spring Boot, so this project is part of my practice and learning journey.
How to Run
1. Clone the project
2. Open in your IDE (IntelliJ / Eclipse)
3. Run the Spring Boot application
4. Open your browser and go to:
http://localhost:8090/swagger-ui.html
Features
Customer management
Item management
Order processing
REST API design
Swagger documentation

Learning Purpose
This project is created for learning purposes and includes important backend concepts like layered architecture, DTO mapping, relationship mapping, and clean API structure.
